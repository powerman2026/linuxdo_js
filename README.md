# linuxdo_js

一个面向 `linux.do` 的 Tampermonkey 用户脚本，用于自动化执行帖子阅读上报，并提供自动点赞、运行参数配置、阅读记录持久化以及夜间随机停机窗口等能力。

> 当前脚本文件：`LINUXDO.js`

## 功能特性

- 自动获取帖子并执行阅读接口上报
- 支持自动点赞，默认仅点赞首楼，可选点赞所有楼层
- 点赞结果增加重试与服务端校验，降低“请求成功但实际未点赞”的概率
- 支持阅读记录持久化，避免重复处理同一主题
- 支持自动循环处理新帖
- 支持从当前浏览位置开始处理
- 支持夜间随机停机窗口，避免凌晨时段继续执行
- 支持通过 Tampermonkey 菜单进行启动、停止、设置和清理记录

## 夜间随机停机规则

脚本内置每日随机停机窗口，并持久化到本地存储：

- 开始时间：`00:00 ~ 00:45` 随机
- 结束时间：`07:00 ~ 08:00` 随机

在停机窗口内：

- 不执行阅读接口
- 不执行自动点赞逻辑
- 脚本暂停
- 若开启 `autoStart`，窗口结束后可自动恢复

## 安装方式

### 方式一：手动安装（推荐）

1. 浏览器安装 `Tampermonkey`
2. 新建脚本
3. 将仓库中的 `LINUXDO.js` 内容完整复制进去
4. 保存后访问 `https://linux.do/*`

### 方式二：从仓库文件导入

1. 打开本仓库中的 `LINUXDO.js`
2. 复制脚本内容到 Tampermonkey 新脚本中
3. 保存并启用脚本

## 使用方法

脚本加载后，可通过 Tampermonkey 菜单看到以下入口：

- `🚀 开始执行`
- `⏹️ 停止执行`
- `⚙️ 设置`
- `🗑️ 清理阅读记录`

建议首次使用时先打开 `⚙️ 设置`，确认以下参数：

- 基础延迟 / 随机延迟
- 单次请求最小 / 最大请求量
- 最小 / 最大阅读时间
- 是否自动运行 `autoStart`
- 是否从当前浏览位置开始 `startFromCurrent`
- 是否自动循环 `autoLoop`
- 是否自动点赞 `autoLike`
- 点赞上限 `likeLimit`
- 是否点赞所有楼层 `likeAllFloors`
- 楼层间点赞间隔 `likeFloorInterval`

## 注意事项

- 本项目当前默认匹配站点为：`https://linux.do/*`
- 脚本依赖页面中的 `csrf token` 与预加载数据完成部分操作
- 自动点赞虽然已增加重试与校验，但仍受站点接口、权限与风控策略影响
- 请仅在你有权限的场景下使用，并遵守目标站点规则
